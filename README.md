# コンパイラ演習 1日目

## セットアップ

### Powershell

```sh
git clone https://github.com/tmu-compiler-info-sys-exp-I/compiler-day1.git
cd compiler-day1
.\bin\setup.bat
```

### UNIX shell (macOS or Linux)

```sh
git clone https://github.com/tmu-compiler-info-sys-exp-I/compiler-day1.git
cd compiler-day1
git clone -b ocaml-4.11.1 --depth 1 https://github.com/felix-lang/win64ocaml.git
```
